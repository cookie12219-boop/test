About this atlas.

Reel Atlas is a coursework project for CS551P Advanced Programming at the University of Aberdeen. It is a database-driven Flask application built around an open movie dataset.

01 Stack

Built with Flask, SQLAlchemy, Jinja2 templates and SQLite. The interface uses pure CSS — no client-side JavaScript, as required by the brief. Tests are written with pytest and run against an in-memory SQLite database.

02 Data

The dataset follows the schema of the TMDB 5,000 Movie Dataset (a public dataset derived from The Movie Database’s public API). The app loads the CSV into two linked tables: a directors table that owns one-to-many movies, and a many-to-many genres table joined through an association table.

03 Code

The application is organised into a factory module, blueprint-based routes, a data loader and a small CLI for refreshing the database. Error handlers cover 400, 404 and 500. See README.md in the repository for installation and run instructions.
